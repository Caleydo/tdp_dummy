=====================

This repository provides a SQLite database with dummy data and additional views for this specific dataset for TargID.

Installation
------------

```
git clone https://github.com/caleydo/targid_dummy.git
cd targid_dummy
npm install
```

Testing
-------

```
npm run test
```

Building
--------

```
npm run build
```
